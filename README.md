## 📖 Project Overview

This project is an educational console application designed to raise **cyber security awareness** in a fun and interactive way. It greets users with a voice message, displays an attractive ASCII logo, personalizes the conversation by name, and teaches practical cyber security tips.

## 🧷 Reason for Creating This Project

In today's digital world, phishing, malware, ransomware, and social engineering remain major threats. Many people still fall victim due to lack of awareness. This chatbot provides **conversational learning** — making cyber security education engaging and easy to understand for students, beginners, and home users.

## ✨ Key Features

- 🎤 **Voice Greeting** on startup (custom WAV file)
- 🖼️ Eye-catching **ASCII Art Logo** with cyber theme
- 👋 **Personalized Interaction** — uses the user's name throughout
- 🌈 **Polished Console UI** with colors, borders, and symbols
- ⌨️ **Typing Animation Effect** for a natural chat feel
- ❓ **Help Menu** with example questions
- 🛡️ **Educational Responses** on important cyber security topics
- ✅ **Input Validation** — handles empty or unknown inputs gracefully
- 🚪 **Exit Command** with a friendly goodbye message
- 🧹 Clean **Object-Oriented Design** (separate classes and methods)
- 🔄 **GitHub Actions CI** for automatic builds

## 📚 Topics Covered

| Topic                        | What the Bot Explains |
|------------------------------|------------------------|
| Strong Passwords & Managers  | Length, complexity, uniqueness, password managers, never reuse |
| Multi-Factor Authentication  | Why MFA is important, types, and best practices |
| Phishing & Spear-Phishing    | How to spot fake emails, links, urgency tricks, and reporting |
| Safe Browsing Habits         | HTTPS, public Wi-Fi risks, VPNs, updates, and downloads |
| Malware & Ransomware         | Common infection methods, prevention, backups, and updates |
| Social Engineering           | Vishing, smishing, and manipulation tactics |
